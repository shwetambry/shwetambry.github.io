    What Projects have I worked on?

    I have undertaken diverse projects such as data cleaning, data analytics, web programming, cloud IaaS, cloud SaaS, network infrastructure set up on cisco routers and switches.

    • Math problems for kids. The main idea of this project is to automate math problem creation for my kids. All problems are automatically graded as well so they know how many problems did they solve correctly. It covers the range of problems such as reading analogue clock, find missing numbers in a sequence of numbers, counting and adding coin denominations, addition, subtraction and multiplication. Moreover, animated help is provisioned for addition, subtraction and multiplication in case they are stuck. I used Javascript, HTML to create it.
